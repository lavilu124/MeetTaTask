## 🚀 Installation & Setup

### Prerequisites
- python 3.13

### Steps to Install
1. **Clone the repository**
   ```bash
   git clone https://github.com/lavilu124/OndaMend.git
   cd OndaMend
   ```
2. **Install dependencies**
   ```bash
   pip3 install flask
   pip3 install pyrebase4
   pip3 install firebase_admin
   ```
3. **Run the app**
   -python3 main.py
